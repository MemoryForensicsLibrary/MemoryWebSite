## MemoryCore Website

Official website and documentation portal for MemoryCore, a low-level C library for live process memory forensics and runtime integrity analysis.
This repository contains the source code for the public website, technical documentation, and API reference of the MemoryCore project.

### Purpose

The website exists to:

- Present the MemoryCore project in a clear and professional way
- Host structured documentation and API references
- Provide installation and usage guides
- Centralize design decisions, architecture explanations, and roadmap updates
- This repository does not contain the core library code.
